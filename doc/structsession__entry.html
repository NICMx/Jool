Detailed Description

A row, intended to be part of one of the session tables. The mapping between the connections, as perceived by both sides (IPv4 vs IPv6).

Field Documentation

struct list_head all_sessions

Chains this session with the rest (see all_sessions, defined in nf_nat_session.h). Used for iterating while looking for expired sessions.

struct bib_entry* bib

Owner bib of this session. Used for quick access during removal. (when the session dies, the BIB might have to die too.)

unsigned int dying_time

Millisecond (from the epoch) this session should expire in, if still inactive.

struct list_head entries_from_bib

Chains this session with the rest from the same BIB (see bib_entry.session_entries). Used by the BIB to know whether it should commit suicide or not.
